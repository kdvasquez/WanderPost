# WanderPost 📬

An iOS app for documenting your 20s — one postcard at a time.

Turn any photo into a personalized digital postcard with your city, a short note, live weather, and a map pin. Save it, share it, or sync it to the cloud.

**[View full project details →](https://kdvasquez.github.io/projects/wanderpost)**

---

## Tech Stack

Swift · SwiftUI · Firebase · MapKit · CoreLocation · OpenWeatherMap API · MVVM

---

## Setup

1. Clone the repo and open `PostcardApp.xcodeproj` in Xcode
2. Add your `GoogleService-Info.plist` from [Firebase Console](https://console.firebase.google.com)
3. Add your OpenWeatherMap API key in `WeatherManager.swift`
4. Press **Cmd + R** to run

> Firebase Storage requires the Blaze (pay-as-you-go) plan as of October 2024.
